# Image Molecule  [<img alt="alt text" src="https://cdn.freebiesupply.com/logos/large/2x/figma-1-logo-png-transparent.png" width="15" />](https://www.figma.com/file/6mNJjcAldqrElBDDFTR8ui/Whitelabel-In-Progress-PRIMARY?node-id=7707%3A343)

Our image molecule is a wrapper for images used in an editorial context, it allows images to have credit text, a caption, and other available metadata.

## Use Cases

-   Displaying photos that need credit shown

-   Displaying photos that have a caption

-   Lead images in articles (ex: Gothamist)

-   In-article images

-   Gallery Images

-  **Don't** use the image molecule if you *just* need an image, use the image atom instead.

 1- **Expand Button** - This icon button should launch a modal of a larger version of the image when clicked. This is an optional feature.

 2- **Caption** - This line of text should describe the picture it’s attached to. Aim to keep the length about the same as a tweet (140 characters). This is optional.

3- **Credit** - The credit line or photo credit identifies the photographer, illustrator, or copyright holder for images in a publication or on a website. These are typically formatted like [artist name] via [publication or source]. This is optional and has flexibility to be optionally hyperlinked. If used, it should always be exposed and not be hidden under the icon button.

4- **Image Atom** - Cropped to set ratio, default 4:3.



## Image Ratios
The image size inside this component comes with an adjustable ratio variable, set at 4:3 by default. Images used inside the component that are not 4:3 will be cropped to fit.



## States

The WNYC variant of this component has 2 states, both related to the caption.

### Caption Hidden
In this state, the caption is hidden. Notice the icon button in the lower right using an info icon.

<img alt="text" src="./images/Molecules-ImageMolcule-CaptionHidden.png" width="400" />

### Caption Visable
Clicking the icon button should change the inside icon from an info icon to a close icon while the caption fades from hidden to visible.
<img alt="text" src="./images/Molecules-ImageMolcule-CaptionVisable.png" width="400" />
